# Airline-Reservation-System-HTML
Simple HTML code for Airline Reservation System

The project consist of simple front end design of an airline reservation system.

#index.html
The home page consist of navigation buttons to diffente pages.

#flightinfo.html
Book A Ticket: This page displays a simple form that gathers information of users who book a ticket.

#rules.html
Rules and Reguations: This page displays the rules to be followed in a flight.

#coupon.html
Offers: This page displays the offers provided by the airline company.

#signup.html
Sign Up / Login: This page helps an user to create an account in the website or to login with an exidting account.
